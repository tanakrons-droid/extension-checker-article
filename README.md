# VSQ CHECKER 🚀

Chrome Extension สำหรับช่วยตรวจสอบความถูกต้องของโครงสร้างบทความ (SEO & UX) โดยเน้นไปที่การตรวจสอบสารบัญ (Table of Contents) และเส้นคั่น (Separator) เหนือหัวข้อ H2

## 📝 คุณสมบัติหลัก (Features)

### 1. ตรวจสอบลิงก์สารบัญ (TOC Anchor Scanning)
- ตรวจสอบว่าลิงก์ในสารบัญทุกลิงก์มี ID ปลายทางอยู่จริงหรือไม่
- แสดงสถานะ **OK** หากลิงก์ใช้งานได้ และ **Broken** หากไม่พบ ID ปลายทาง
- รองรับการถอดรหัส URL (Decoded URI)

### 2. ตรวจสอบเส้นคั่นหัวข้อ (H2 Separator Scanning)
- ตรวจสอบว่าเหนือหัวข้อ H2 มีเส้นคั่น (`<hr>` หรือ Element ที่มี class เกี่ยวกับ separator/line/divider) หรือไม่
- **ระบบกรองอัจฉริยะ (Smart Filtering):**
    - ⏭️ **ข้าม H2 ตัวแรกสุด** ของเนื้อหา
    - ⏭️ **ข้าม H2 ที่เป็นส่วนหนึ่งของสารบัญ**
    - ⏭️ **หยุดการตรวจสอบเมื่อถึงส่วน Footer ของบทความ** เช่น ส่วนของ "อ้างอิง (References)", "บทความแนะนำ", หรือ "Related Articles" เพื่อลดความผิดพลาด (False Positive)

### 3. ตรวจสอบเส้นคั่นผิดตำแหน่ง (Misplaced Separator Scanning)
- ตรวจสอบกรณีที่มีการใช้เส้น separator เหนือหัวข้อที่ไม่ใช่ H2 (เช่น อยู่เหนือ H3, H4 หรือ Paragraph ทั่วไป)
- ระบบจะแจ้งเตือนเป็น **Wrong Position** เพื่อให้ทราบว่ามีการวางโครงสร้างผิดตำแหน่ง

### 4. การโต้ตอบและการตั้งค่า (Interaction & Settings)
- **Highlight on Click:** คลิกที่รายการในผลลัพธ์เพื่อเลื่อน (Scroll) ไปยังตำแหน่งนั้นบนหน้าเว็บ พร้อมมีเอฟเฟกต์สีเน้น (Pulse effect)
- **Custom Selectors:** สามารถเพิ่ม/แก้ไข CSS Selector ของสารบัญให้รองรับกับหลายๆ เว็บไซต์หรือปลั๊กอิน TOC ต่างๆ (เช่น Easy TOC, Rank Math, LuckyWP เป็นต้น)
- **Auto Detect:** ระบบค้นหาสารบัญอัตโนมัติจาก Selector ที่กำหนดไว้
- **Copy Report:** ปุ่มคัดลอกสรุปผลการตรวจสอบเพื่อนำไปส่งงานหรือเก็บเป็น Log

## 🛠 วิธีติดตั้ง (Installation)

1. ดาวน์โหลดหรือ Clone โปรเจกต์นี้ลงในเครื่องคอมพิวเตอร์ของคุณ
2. เปิดเบราว์เซอร์ Google Chrome หรือเบราว์เซอร์ที่ใช้ Chromium
3. พิมพ์ `chrome://extensions/` ในช่อง Address bar
4. เปิดโหมดนักพัฒนา (**Developer mode**) ที่มุมขวาบน
5. คลิกปุ่ม **Load unpacked** (โหลดส่วนขยายที่คลายแพ็กแล้ว)
6. เลือกโฟลเดอร์ของโปรเจกต์นี้

## 🚀 วิธีใช้งาน (How to Use)

1. เข้าไปยังหน้าเว็บไซต์หรือบทความที่ต้องการตรวจสอบ
2. คลิกที่ไอคอน **VSQ CHECKER** บนแถบเครื่องมือของ Chrome
3. เลือกตั้งค่าที่ต้องการ (เช่น แสดงเฉพาะลิงก์ที่เสีย หรือ ใส่ Selector เพิ่มเติม)
4. กดปุ่ม **Scan Website**
5. ตรวจสอบผลลัพธ์ในหน้า Popup:
    - สีเขียว = ปกติ
    - สีแดง = พบปัญหา (ลิงก์เสีย หรือ ลืมใส่เส้นคั่น)
6. คลิกที่รายการเพื่อตรวจสอบตำแหน่งจริงบนเว็บไซต์
